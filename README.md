# Entwickler-Portfolio

Vielen Dank für Ihr Interesse an meinen Projekten.

Dieses Repository enthält eine Auswahl meiner privaten Entwicklungsprojekte mit Schwerpunkt auf Desktop-Anwendungen mit einer sauberen Architektur. Hintergrund für den Beginn dieser Projekte ist meine Leidenschaft für die Softwareentwicklung. Damit verbunden ist ein großes Interesse dafür, mich stets weiterzuentwickeln und meine bestehenden Fähigkeiten und Kenntnisse zu vertiefen.

Der Fokus einiger Projekte liegt auf praxisnaher Software mit realistischen Anforderungen.


## Projektübersicht

### 1. Rechnungsverwaltung – Desktopanwendung
Umfangreiche Desktop-Software zur Erstellung und Verwaltung von Rechnungen
inklusive PDF-Export sowie Kunden- und Produktverwaltung.

**Schwerpunkte:**
- MVVM-Architektur
- Dependency Injection
- Persistente Speicherung ([SQLite][SQLite-url], Entity Framework)
- Benutzerfreundliches UI & Dashboard

→ Details: [`Projekte/UnRechnung/`](Projekte/UnRechnung/)


### 2. yt-dlp GUI – Desktopanwendung
Grafische Benutzeroberfläche zur komfortablen Nutzung einer Konsolenanwendung inklusive Download-Queue, Fortschrittsanzeige und Prozesssteuerung.

**Schwerpunkte:**
- Prozesssteuerung
- Asynchrone Verarbeitung
- Verarbeitung von Prozessausgaben und Statusinformationen
- UI-Status- und Fortschrittsdarstellung

→ Details: [`Projekte/YouTube-Downloader/`](Projekte/YouTube-Downloader/)


### 3. Advent of Code – Programmierherausforderungen
Sammlung algorithmischer Aufgaben zur Vertiefung von Problemlösungsfähigkeiten
und dem sicheren Umgang mit Datenstrukturen und Algorithmen.

**Schwerpunkte:**

- Algorithmisches Denken
- Datenstrukturen
- Effizienz & Laufzeit
- Sauber strukturierte Lösungen

→ Details: [`Projekte/AdventOfCode/`](Projekte/AdventOfCode/)


## Technologiestack

- Sprache: C#
- UI: [Avalonia UI][Avalonia-url]
- Architektur: MVVM
- Datenbanken: [SQLite][SQLite-url]
- ORM: Entity Framework
